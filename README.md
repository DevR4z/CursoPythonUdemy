## 🚀 Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/DevR4z/CursoPythonUdemy.git
```

2. Acesse a pasta desejada:
```bash
cd CursoPythonUdemy/Exercicios
```

3. Execute qualquer arquivo Python:
```bash
python nome_do_arquivo.py
```

> **Requisito:** Python 3.x instalado. [Download aqui](https://www.python.org/downloads/)
